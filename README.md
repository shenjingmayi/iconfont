### **📦 脚本主要功能**

本脚本用于一键下载iconfont图标合集里的**所有**图标（默认保存为SVG格式）

截止至1.0版本（2025年2月16日）仅实现了图标合集下载，矢量插画库、3D插画库、动效库、字体库暂且不支持一键下载。
